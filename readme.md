# mac-recorder-app

A cli app that records audio from mic or your macbook, and stored it in a .wav file.

## Requirements
- Rust setup in your local
- Cargo package manager.

## Steps to run
- Clone the repo
  > git clone https://github.com/anubhavitis/mac-recording-app
- ```cargo run```
